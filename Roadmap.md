# 🗺️ DataFactory 数字化工厂 - MLOps 演进路线图

## 🏗️ 阶段一：标准化与精细化生产 (v1.x)
- [x] **环境变量管理**：敏感信息（邮件密码、路径）已移至 `.env`。
- [ ] **精细化筛选 (Current Focus)**：实现“单兵作战”，精准识别并只量产合格物料，踢除次品。
- [ ] **自动化产量日报**：
    - [ ] **批次总结报告**：每个 Batch 结束后，自动生成包含合格率、指纹统计、存储位置的详细邮件。
    - [ ] **定期巡检播报**：建立“心跳机制”，每 4 小时向厂长推送一次生产状态（处理了多少视频、报错次数、产线空闲/忙碌）。
- [ ] **日志系统升级**：引入 `logging` 模块，记录生产流水日志，支持按级别查看错误。

## 📊 阶段二：实验跟踪与数据血缘 (v2.x)
- [ ] **MLflow Tracking**：记录模型参数（门槛、时长）与质量指标。
- [ ] **DVC 数据版本化**：实现原始物料与产出数据的血缘追踪。
- [ ] **ML模型接入**：将硬编码规则质检升级为 CNN 或异常检测模型。

## 🐳 阶段三：容器化与云原生 (v3.x)
- [ ] **Docker 容器化**：实现环境一键部署。
- [ ] **Prometheus + Grafana**：搭建实时可视化仪表盘，展示生产效率曲线。
- [ ] **GitHub Actions**：自动化 CI/CD 测试流水线。

## 🚀 阶段四：真实生产环境适配 (v4.x)
- [ ] **大规模吞吐测试**：对接实验室真实标注数据。
- [ ] **性能白皮书**：量化成果，总结系统在真实工业场景下的表现。
